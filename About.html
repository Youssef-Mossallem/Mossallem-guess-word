🧠 About the Game

“Mossallem Guess Word” is a word guessing game created using HTML, CSS, and JavaScript. The goal is to guess the hidden word by entering letters.

The game provides hints and color feedback:

  • blue: Letter is correct and in the right place
  • Dark green: Letter is correct but in the wrong place
  • Red: Letter is wrong
You can challenge your memory, vocabulary, and logic skills while having fun!

“Mossallem Guess Word” هي لعبة لتخمين الكلمات تم إنشاؤها باستخدام HTML و CSS و JavaScript. الهدف هو تخمين الكلمة المخفية عن طريق إدخال الحروف.

اللعبة توفر تلميحات وألوان توضح النتيجة:

  • ازرق: الحرف صحيح وفي المكان الصحيح
  • اخضر غامق: الحرف صحيح ولكن في المكان الخطأ
  • احمر: الحرف غير صحيح
يمكنك من خلالها تحدي ذاكرتك ومهاراتك في اللغة والمنطق أثناء الاستمتاع!
